Coloca aquí todos los archivos CSS originales migrados desde el portafolio clásico (por ejemplo: menu.css, menu-header.css, menu-title.css, etc.).

Para usar estos estilos en tu Next.js, importa los archivos necesarios en src/app/globals.css usando:
@import '../styles/nombre-del-archivo.css';

Esto mantiene el código organizado y facilita la migración por componentes.
